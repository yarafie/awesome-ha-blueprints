# Library Documentation Mirror

This directory contains the website-facing documentation mirror of the
Awesome HA Blueprints Library.

Source of truth:

- `/library`

Rules:

- Directory structure mirrors `/library` closely (1:1 where possible)
- Content here is documentation-oriented (MDX/Markdown)
- No canonical data (YAML/JSON assets) should be authored here

This directory is consumed by the website (Docusaurus).
