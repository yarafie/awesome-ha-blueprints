## Description

This blueprint provides universal support for running any custom action when an event is detected on the provided Aqara DJT11LM Vibration Sensor.
Supports controllers integrated with Zigbee2MQTT.
The blueprint will automagically detect the integration type and take care of all the rest.

:::tip
Automations created with this blueprint is not connected to any [Hooks](/docs/blueprints/hooks).
:::

## Additional Notes

### Device Sensitivity

Please note that although this blueprint provides support for setting the sensitivity of the device to High [1], Medium [11] or Low [21].

According to Zigbee2MQTT documentation [Zigbee2MQTT](https://www.zigbee2mqtt.io/devices/DJT11LM.html#sensitivity) After setting the sensitivity you immediately have to start pressing the reset button with an interval of 1 second until you see Zigbee2MQTT publishing the new sensitivity to MQTT.
