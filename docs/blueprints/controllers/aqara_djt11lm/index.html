Descriptionâ€‹

This blueprint provides universal support for running any custom action when an event is detected on the provided Aqara DJT11LM Vibration Sensor. Supports controllers integrated with Zigbee2MQTT. The blueprint will automagically detect the integration type and take care of all the rest.

tip

Automations created with this blueprint is not connected to any Hooks.
